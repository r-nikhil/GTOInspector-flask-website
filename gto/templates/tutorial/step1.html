Uploading hands

GTO Inspector currently support Pokerstars.com format hand histories which can be procured from trackers like PT4, HM3 etc, Adda52, GGPoker hand histories.

For exporting hand history from PT4, follow the steps below:

Step 1: Export Hands from PT4

○ Filter and select the hands(ctrl+a and right click) you want to be analysed. Right click anywhere on the selected hands and click “Export Original Hand Histories”

Step 2: Choose a location to save the file. Remember this is the file which has to be uploaded on the website

○ For help with other trackers please visit their documentation site or contact us on gtoinspector.poker@gmail or join our discord server here

Step 3: Check for processing status on the “Upload History” page

○ Once your report is generated you would get three options to choose from. You can click “View Analysis Dashboard” to get redirected to the dashboard, “Download HH File” to download the original .txt file uploaded by the user and “Download Processed CSV Report” to download the .csv report generated by GTOInspector.

○ The full report has the following columns: Hand/Pairedness/Suitedness/Hand Category/Position/Result/Opportunity/Stack Size/Big Blind/Player's Move/GTO Move/Amount Won in Terms of BB/Move EV/GTO EV/Correct Moves EV/Wrong Moves EV/Cumulative Mistakes EV/Cumulative Correct Moves EV/Cumulative Earnings/Cumulative GTO EV/EV Difference
